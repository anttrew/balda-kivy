# Балда на Python/Kivy

Небольшая Android-игра «Балда» для 2–4 игроков. Проект сделан так, чтобы его можно было запустить на компьютере, залить на GitHub и собрать APK через GitHub Actions.

## Что уже реализовано

- главное меню: новая игра, продолжить, настройки;
- выбор размера поля 5×5 или 7×7;
- выбор 2–4 игроков и их имён;
- стартовое слово по центру поля;
- локальное сохранение партии;
- подсветка стартового слова светло-серым;
- подсветка новой буквы тёмно-серым;
- удаление новой буквы до отправки слова;
- составление слова тапами по соседним клеткам;
- запрет диагоналей;
- запрет повторного использования клетки в одном слове;
- слово обязательно должно содержать новую букву;
- запрет повторных слов;
- список слов по каждому игроку;
- подсчёт очков по длине слов;
- экран итогов с пьедесталом первых трёх мест;
- простые настройки: строгий словарь и минимальная длина слова;
- GitHub Actions workflow для сборки APK.

## Важное про словарь

Файл `data/words_ru.txt` — это мини-словарь для тестирования. Он нужен, чтобы работала кнопка случайного стартового слова и опциональная строгая проверка.

Для настоящей игры лучше заменить его большим словарём: одно слово на строку, кодировка UTF-8. Если строгая проверка выключена, приложение проверяет только повторы слов, а корректность слов игроки контролируют сами.

## Структура проекта

```text
balda_kivy/
├── main.py                         # Kivy UI и экраны приложения
├── buildozer.spec                  # настройки Android-сборки
├── requirements.txt                # зависимости для запуска на компьютере
├── data/words_ru.txt               # локальный словарь
├── balda/
│   ├── dictionary.py               # нормализация и проверка слов
│   ├── engine.py                   # правила игры
│   ├── models.py                   # состояние игры, игроки, клетки
│   └── storage.py                  # сохранение JSON
├── tests/test_engine.py            # базовые тесты правил
└── .github/workflows/build-apk.yml # сборка APK на GitHub
```

## Локальный запуск на компьютере

### Linux/macOS

```bash
cd balda_kivy
python3 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
pip install -r requirements.txt
python main.py
```

### Windows

Для простого запуска на компьютере можно использовать обычный Python:

```powershell
cd balda_kivy
py -m venv .venv
.\.venv\Scripts\activate
python -m pip install --upgrade pip
pip install -r requirements.txt
python main.py
```

Для Android-сборки на Windows лучше использовать WSL2 Ubuntu.

## Локальная сборка APK через WSL/Linux

Установите системные зависимости:

```bash
sudo apt update
sudo apt install -y git zip unzip openjdk-17-jdk python3-pip python3-venv autoconf libtool pkg-config zlib1g-dev libncurses5-dev libncursesw5-dev libtinfo6 cmake libffi-dev libssl-dev
```

Подготовьте окружение:

```bash
cd balda_kivy
python3 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
pip install buildozer cython
```

Соберите APK:

```bash
buildozer -v android debug
```

APK появится в папке `bin/`.

Подключите Android-телефон по USB, включите режим разработчика и USB debugging, затем можно поставить и запустить так:

```bash
buildozer -v android debug deploy run logcat
```

## Сборка APK на GitHub

1. Создайте пустой репозиторий на GitHub, например `balda-kivy`.
2. В локальной папке проекта выполните:

```bash
git init
git add .
git commit -m "Initial Balda Kivy app"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/balda-kivy.git
git push -u origin main
```

3. Откройте репозиторий на GitHub.
4. Перейдите во вкладку **Actions**.
5. Откройте workflow **Build Android APK**.
6. После успешной сборки внизу страницы workflow run появится artifact **balda-debug-apk**.
7. Скачайте artifact, распакуйте ZIP, внутри будет APK.

## Более удобный вариант для родителей: GitHub Release

Workflow также публикует APK в GitHub Release, если отправить git-tag, начинающийся с `v`:

```bash
git tag v0.1.0
git push origin v0.1.0
```

После сборки откройте страницу репозитория → **Releases** → `v0.1.0`. Там будет APK-файл. Его уже проще отправить родителям ссылкой.

## Установка APK на телефон

1. Скачайте APK на Android-телефон.
2. Откройте файл APK.
3. Android может попросить разрешить установку из этого источника, например для Chrome или файлового менеджера.
4. Включите разрешение и подтвердите установку.

Ставьте APK только из своего репозитория или из файла, который вы сами собрали.

## Игровой процесс

1. На экране новой игры выберите размер поля и игроков.
2. Укажите стартовое слово длиной в размер поля.
3. Тапните пустую клетку рядом с уже заполненной клеткой.
4. Введите одну русскую букву.
5. Тапами выберите цепочку букв для слова.
6. Нажмите **Готово**.
7. Если буква не подошла, нажмите **Удалить букву**.
8. Когда поле заполнится, появится экран итогов.
